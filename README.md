# Playwright Projects

Repositório para centralizar projetos de testes **End-to-End (E2E)** com [Playwright](https://playwright.dev/).

## Projetos atuais

| Projeto | Descrição |
|---|---|
| `swaglabs-testing` | Testes E2E no SauceDemo |
| `swaglabs-testing-pom` | Testes E2E no SauceDemo com Page Object Model (POM) |

## Como usar

```bash
cd <nome-do-projeto>
npm install
npx playwright install
npx playwright test
```

## Tecnologias

- **Playwright** — Automação de navegadores
- **JavaScript** — Linguagem dos testes
- **Allure** — Relatórios avançados (em alguns projetos)
